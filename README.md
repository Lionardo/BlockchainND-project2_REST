# Poject2 RESTful WEB API

this is the restful api project for the blockchain nanodegree project.

## Framework

Express.js
with the help of express generator in this project
https://expressjs.com/

### Prerequisites

```
npm
express.js
nodemon
```

### Installing

1- npm i 
2- npm run start


## Enpoints

>localhost:8000/block

post enpoint to add a new block with data params like:
```
{
      "body": "Testing block with test string data"
}
```



> localhost:8000/block/:number

get enpoint to fetch the block with height number
